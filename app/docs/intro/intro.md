# Curso de TI para Controladoria

:::tip Imagine que...
Seu chefe pede: "Me traga o total de vendas por cliente dos últimos 3 meses, com a margem de cada produto e a comparação com o mesmo período do ano passado."

No seu dia a dia, isso significa: pedir pro TI extrair os dados, esperar dias, receber uma planilha gigante, e passar horas no Excel com PROCV, tabela dinâmica e fórmulas manuais.

Agora imagine que você mesmo pode abrir o sistema, digitar uma pergunta em SQL e ter a resposta em **segundos**. É isso que você vai aprender aqui: **autonomia sobre os dados**.
:::

Bem-vindo ao curso que vai transformar sua expertise em **controladoria, fiscal e contabilidade** em superpoderes com **tecnologia de dados**.

## Para Quem é Este Curso

Você já domina: balanços, DRE, fluxo de caixa, impostos (PIS, COFINS, ICMS, IRPJ, CSLL), plano de contas, conciliação contábil, e todas as rotinas de fechamento.

Agora você vai aprender a **extrair, transformar, visualizar e analisar dados** usando as ferramentas que o mercado exige.

## O Que Você Vai Aprender

| Módulo | Tema | Carga |
|--------|------|-------|
| 1 | **SQL** — Consultas, agregações, joins, window functions | 16h |
| 2 | **Google BigQuery** — Data warehouse na nuvem | 10h |
| 3 | **Looker** — BI com LookML | 8h |
| 4 | **Tableau** — Visualização de dados | 8h |
| 5 | **IA Aplicada** — Machine learning para finanças | 10h |
| 6 | **Projeto Final** — Pipeline completa | 6h |

:::note Por que isso importa para você?
Na controladoria, informação é poder. Quanto mais rápido você acessa os dados, mais rápido fecha o balanço, responde ao auditor ou apresenta aquele relatório que a diretoria pediu de última hora. Este curso não é sobre virar programador — é sobre **você não depender mais de ninguém para buscar informações**. Você continua sendo contador, mas com o superpoder de acessar qualquer dado em segundos.
:::

## Como Usar o Curso

- Navegue pelos módulos na barra lateral
- Use o **SQL Playground** (link na navbar) para praticar queries em tempo real
- Todos os exemplos usam o banco de dados fictício do **Grupo Nova Era S.A.**

## O Banco de Dados Nova Era

O banco simula uma empresa industrial brasileira com 3 subsidiárias:

| Tabela | Descrição |
|--------|-----------|
| `empresas` | Grupo Nova Era (matriz + 2 filiais) |
| `planos_contas` | Plano de contas completo (níveis 0-3) |
| `centros_custo` | 10 centros de custo (ADM, FIN, COM, PROD, LOG, TI, RH, JUR) |
| `lancamentos_contabeis` | Partidas dobradas (débito/crédito) de Jan a Jun/2026 |
| `clientes` | 10 clientes em 5 estados |
| `fornecedores` | 9 fornecedores de matéria-prima, serviços, utilidades |
| `faturamento` | 35 notas fiscais de venda |
| `contas_receber` | Contas a receber com status (aberto/recebido) |
| `contas_pagar` | Contas a pagar com status (aberto/pago) |
| `notas_fiscais` | NF de entrada e saída com base e valor de imposto |
| `funcionarios` | 15 funcionários com salários e cargos |
| `folha_pagamento` | 27 registros de folha mensal |
| `dre_mensal` | DRE simplificada por mês (jan-jun/2026) |
| `impostos_recolher` | PIS, COFINS, ICMS apurados mensalmente |
| `produtos` | 8 produtos com preço e custo unitário |

## Pré-requisitos

- Conhecimento sólido de contabilidade e controladoria
- Navegador web moderno (Chrome, Firefox, Edge)
- **Nada para instalar** — tudo roda no navegador

## Para fixar

- Este curso ensina tecnologia **para profissionais de controladoria** — não para programadores
- Você vai aprender a buscar dados sozinho, sem depender do TI
- Tudo roda no navegador: **nada para instalar**
